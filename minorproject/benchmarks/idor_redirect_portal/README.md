# Northstar Online Benchmark

Small Node.js customer portal for local benchmark scans.

## Run

```powershell
npm start
```

The app starts on `http://localhost:4002` by default. Set `PORT` to use another port.

## Useful Pages

- `/`
- `/dashboard?user=101`
- `/statements?user=101`
- `/statements/7001?user=101`
- `/continue?next=https://www.example.com/rewards`
